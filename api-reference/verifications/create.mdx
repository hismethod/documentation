---
title: "Create Verification"
openapi: "openapi.json POST /v1/verifications"
---
Creates a new identity verification session.

### Request Body

| Parameter             | Type   | Required | Description                                                                                                          |
| :-------------------- | :----- | :------- | :------------------------------------------------------------------------------------------------------------------- |
| `eid`                 | string | **Yes**  | The ID of one of the [**Supported eIDs**](/core-concepts/supported-eids).                                            |
| `redirectUri`         | string | No       | The URL to redirect to after a `redirect` flow completes. Recommended for `qr` flows as well for the final redirect. |
| `webhookUrl`          | string | No       | The webhook URL to receive status change notifications.                                                              |
| `userData`            | object | No       | User information required by some eIDs (e.g., `evrotrust`, `smartid`).                                               |
| `codeChallenge`       | string | No       | The Code Challenge for a PKCE flow.                                                                                  |
| `codeChallengeMethod` | string | No       | The PKCE method. `S256` (recommended) or `plain`.                                                                    |

### cURL Examples

<CodeGroup>

```bash BankID SE (QR)
curl --request POST \
  --url https://api-gw.hopae.com/v1/verifications \
  --user "YOUR_CLIENT_ID:YOUR_CLIENT_SECRET" \
  --header "Content-Type: application/json" \
  --data '{
    "eid": "bankidse"
  }'
```


```bash MitID (Redirect + PKCE)
curl --request POST \
  --url https://api-gw.hopae.com/v1/verifications \
  --user "YOUR_CLIENT_ID:YOUR_CLIENT_SECRET" \
  --header "Content-Type: application/json" \
  --data '{
    "eid": "mitid",
    "redirectUri": "https://yourapp.com/auth/callback",
    "codeChallenge": "E9Melhoa2OwvFrEMTJguCHaoeK1t8URWbuGJSstw-cM",
    "codeChallengeMethod": "S256"
  }'
```


```bash Evrotrust (Push + userData)
curl --request POST \
  --url https://api-gw.hopae.com/v1/verifications \
  --user "YOUR_CLIENT_ID:YOUR_CLIENT_SECRET" \
  --header "Content-Type: application/json" \
  --data '{
    "eid": "evrotrust",
    "userData": {
      "phone": "+359881234567"
    }
  }'
```
